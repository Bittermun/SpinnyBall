# Technical Specification — Spin-Stabilized Gyroscopic Mass-Stream Anchor

## System Architecture

The anchor system suspends a station node via a closed-loop, counter-propagating stream of spin-stabilized projectile packets. Stiffness is generated by momentum-flux reaction and augmented by flux-pinning superconducting bearings. A feedback controller (gain `g_gain`) modulates the effective stiffness in real time.

## Physical Model

### Effective Stiffness

```
k_eff = lam * u^2 * g_gain + k_fp
```

| Configuration | k_eff (N/m) | Target Range (N/m) | Status |
|---------------|-------------|-------------------|--------|
| Operational (1600 m/s) | 11,973 | 6,000–10,000 | 20% above |
| Optimal (Sobol, 589 m/s) | 8,145 | 6,000–10,000 | Within |
| Paper target | 8,000 | 6,000–10,000 | Within |

Recommendation: Use optimal Sobol parameters for paper claims — reduces momentum flux 8x while remaining within all physical margins.

### Momentum Flux

```
F = lam * u^2
```

| Configuration | F (N) | Expected Range (N) |
|---------------|-------|-------------------|
| Operational | 4.27e7 | 1e3 – 1e7 |
| Optimal | 5.36e6 | Within range |

### Centrifugal Stress

```
sigma = rho * r^2 * omega^2   [rho = 2500 kg/m^3 BFRP]
```

| Spin Rate | RPM | Stress (MPa) | Margin |
|-----------|-----|--------------|--------|
| 5,236 rad/s | 50,000 | 685.4 | 14.3% (below 20% threshold) |
| 4,000 rad/s | 38,197 | 400.0 | 50.0% |
| 5,657 rad/s | 54,019 | 800.0 | At limit |

Note: Reducing to 40,000 RPM brings stress margin to 50%.

### Radiative Thermal Limit

```
T_ss = (P / (eps * A * sigma_SB))^0.25
```

- Heating power: 200 W (eddy current + solar)
- Emissivity: 0.85, Surface area: 0.2 m^2
- Result: T_ss = 379.5 K — 15.7% below 450 K limit

## Statistical Methodology

| Element | Implementation |
|---------|---------------|
| Confidence intervals | Wilson score (binomial); normal CI (means) |
| Convergence criterion | Wilson CI width < 5% of range [0,1] |
| Adaptive MC strategy | Start N=100, double until converged or N > 10,000 |
| Fault injection | fault_prob_per_step = fault_rate * dt / 3600 |
| Containment threshold | 2 nodes |
| Cascade threshold | 1.05x stiffness reduction |

Convergence: N=100 achieves CI width of 3.7% — converged. N=20 (quick profile sweep) achieves 15% — preliminary only.

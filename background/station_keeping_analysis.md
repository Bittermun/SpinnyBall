## Physics Background

### Momentum-Flux Force Law

The anchoring force is generated by deflecting a counter-stream of mass packets:

$$F = \lambda u^2 \theta$$

Where:
- $\lambda$ = linear mass density (kg/m)
- $u$ = relative velocity (m/s)
- $\theta$ = deflection angle (radians)

### Control Stiffness

With proportional feedback control:

$$k_{\text{control}} = \lambda u^2 g_{\text{gain}}$$

Where $g_{\text{gain}}$ is the controller gain (dimensionless, typically 0.05–0.8).

### Natural Frequency

For a station of mass $m_s$:

$$\omega_n = \sqrt{\frac{k_{\text{control}}}{m_s}}$$

$$T = \frac{2\pi}{\omega_n}$$

## Energy Efficiency Proof

### Drag Braking Energy Loss

To generate lateral force $F$ via drag braking:

$$\Delta v = \frac{F}{\lambda}$$

$$P_{\text{drag}} = \frac{1}{2} \lambda u (\Delta v)^2 = \frac{F^2 u}{2 \lambda}$$

For $F = 1000$ N, $\lambda = 5$ kg/m, $u = 100$ m/s:

$$P_{\text{drag}} = \frac{(1000)^2 \times 100}{2 \times 5} = 10,000,000 \text{ W} = 10 \text{ MW}$$

### Momentum-Flux Energy Cost

Momentum-flux steering redirects packets **without slowing them**, so the only energy cost is:
- Superconducting coil power (persistent current, ~0 W ideal)
- Cryocooler overhead (~1–10 W for small system)
- Control electronics (~0.1–1 W)

**Total: ~10 W**

### Efficiency Ratio

$$\text{Efficiency} = \frac{P_{\text{drag}}}{P_{\text{control}}} = \frac{10,000,000}{10} = 1,000,000\times$$

**This is the core claim:** Momentum-flux anchoring is **orders of magnitude more energy-efficient** than drag braking.

## Appendix: Quick Reference Calculations

```python
import math

def anchor_stiffness(lambda_mass, u, g_gain):
    """Calculate control stiffness from momentum-flux anchor."""
    return lambda_mass * u**2 * g_gain

def natural_frequency(k, ms):
    """Calculate natural frequency of station."""
    return math.sqrt(k / ms)

def drag_power(F, lambda_mass, u):
    """Calculate power loss from drag braking."""
    delta_v = F / lambda_mass
    return 0.5 * lambda_mass * u * delta_v**2

# Example usage
k = anchor_stiffness(5.0, 100.0, 0.8)  # 40,000 N/m
omega = natural_frequency(k, 10000.0)  # 2.0 rad/s
P_drag = drag_power(1000.0, 5.0, 100.0)  # 10 MW
```
